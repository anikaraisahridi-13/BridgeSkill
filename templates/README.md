BridgeSkill

BridgeSkill is a mentorship platform that connects students with industry professionals to guide their career journey.

Features
Student and Mentor Authentication
Mentor Request System
Accept / Reject Requests
Separate Dashboards for Students and Mentors
Tech Stack
Backend: Flask (Python)
Database: MySQL
Frontend: HTML, CSS, JavaScript
Project Structure
templates/ → HTML files
static/ → CSS and JavaScript files
app.py → Main Flask application
models.py → Database models
database.py → Database connection
How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open in browser:

http://127.0.0.1:5000/
